## 🎯 نظرة عامة

**Civilun AI** هي منصة ذكاء اصطناعي متطورة مصممة خصيصًا للمهندسين المدنيين والتقنيين في مجال البناء والتشييد. تجمع المنصة بين قوة النماذج اللغوية الكبيرة (LLM) والخبرة الهندسية المتخصصة لتقديم مساعد ذكي قادر على:

- تحليل الأكواد الهندسية والمواصفات التقنية
- مراجعة المستندات الإنشائية والمخططات
- إجراء الحسابات الهندسية المعقدة
- إدارة وثائق المشاريع
- تقديم استشارات هندسية فورية

### لماذا Civilun؟

| الميزة | الوصف |
|---------|-------|
| 🎯 **تخصص هندسي** | مبني من قبل مهندسين ولمهندسين مدنيين |
| 🧠 **ذكاء متعدد** | دعم أكثر من 10 مزودي ذكاء اصطناعي |
| 📚 **معرفة شاملة** | قاعدة بيانات هندسية ضخمة مع RAG |
| 🔒 **أمان عالي** | تشفير من الدرجة العسكرية وحماية البيانات |
| 🌍 **دعم عربي كامل** | واجهة وSupportedContent باللغة العربية |
| ⚡ **أداء فائق** | استجابة فائقة السرعة مع الكاش الذكي |

## ✨ المميزات الرئيسية

### 🤖 نظام الذكاء الاصطناعي المتقدم

- **دعم متعدد النماذج**: يدعم أكثر من 30 نموذج ذكاء اصطناعي من مزودين عالميين:
  - OpenAI (GPT-4o, GPT-4o Mini)
  - Groq (Llama 3.3 70B, Llama 3.1 8B)
  - Google (Gemini 2.0 Flash)
  - Anthropic (Claude 3.5 Sonnet)
  - DeepSeek (V3, R1)
  - OpenRouter (وصول لأكثر من 100 نموذج)
  - NVIDIA API
  - Moonshot AI (Kimi)
  - HuggingFace

- **اختيار تلقائي ذكي**: نظام يختار أفضل نموذج متاح بناءً على:
  - جودة النموذج (Tier 1, 2, 3)
  - توفر مفاتيح API
  - سرعة الاستجابة
  - التكلفة
  - دعم الصور (Vision)

- **تحويل تلقائي عند الفشل**: إذا فشل نموذج، ينتقل تلقائيًا للنموذج التالي

### 📚 نظام RAG المتقدم (Retrieval-Augmented Generation)

- **فهرسة ذكية**: تقسيم المستندات بطريقة هرمية تحافظ على السياق
- **بحث دلالي**: استخدام التضمينات (Embeddings) للبحث بالمعنى
- **إعادة ترتيب ذكية**: Reranking لتحسين دقة النتائج
- **كاش دلالي**: تخزين مؤقت للأسئلة المتكررة (أقل من 200ms)
- **دعم ملفات متعددة**: PDF, DOCX, TXT, XLSX, PPTX, HTML, CSV, وأكواد برمجية

### 🔧 أدوات هندسية متخصصة

- **تحليل إنشائي**: تحليل الكمرات والإطارات باستخدام AnaStruct
- **حساب الكميات**: حصر خرسانة وحديد وقوالب
- **تحليل القطاعات**: تحليل القطاعات الخرسانية المسلحة
- **حسابات جيوتكنيك**: قدرة تحمل الأساسات السطحية
- **تحويل مساحي**: تحويل الإحداثيات بين الأنظمة (WGS84, UTM)
- **حل معادلات**: حل رمزي للمعادلات الهندسية باستخدام SymPy
- **تحليل BOQ**: استخراج وتحليل جداول الكميات
- **مراجعة المواصفات**: تحليل المواصفات الفنية والتعارضات
- **فحص الشروخ**: تحليل صور العيوب الإنشائية (Vision)
- **تحليل Shop Drawings**: مراجعة اللوحات التنفيذية
- **صياغة RFI**: تحويل التعارضات إلى طلبات معلومات

### 💬 نظام المحادثة الذكي

- **محادثات غير محدودة**: مع دعم السياق الطويل
- **تلخيص تلقائي**: ضغط المحادثات الطويلة تلقائيًا
- **ذاكرة محادثة**: الحفاظ على سياق المحادثة
- **دعم المرفقات**: رفع صور وملفات PDF ومستندات
- **بحث ويب**: البحث التلقائي على الإنترنت عند الحاجة
- **أسئلة توضيحية**: نظام ذكي لطلب المعلومات الناقصة
- **تصدير المحادثات**: تصدير إلى PDF أو DOCX

### 🎨 واجهة مستخدم متقدمة

- **تصميم عصري**: واجهة مستخدم حديثة مع تأثيرات حركية
- **دعم الوضع الداكن/الفاتح**: تبديل سهل بين الثيمات
- **ألوان قابلة للتخصيص**: 6 ألوان رئيسية (تيل، بنفسجي، أزرق، برتقالي، وردي، أخضر)
- **تصميم متجاوب**: يعمل على جميع الأجهزة
- **دعم RTL كامل**: واجهة عربية بالكامل
- **تأثيرات جزيئية**: خلفية متحركة بتأثير Neural Particles

## 🖥️ واجهات المستخدم

### الصفحة الرئيسية (Landing Page)

الصفحة الرئيسية مصممة بعناية لتقديم المنصة بشكل احترافي:

- **قسم Hero**: عنوان جذاب مع إحصائيات حية وتأثيرات حركية
- **مؤشرات الثقة**: شهادات الأمان والامتثال (SOC 2, GDPR)
- **قسم المميزات**: عرض الميزات الرئيسية بشكل تفاعلي
- **كيف يعمل**: شرح ثلاثي الخطوات لكيفية الاستخدام
- **من نحن**: معلومات عن الفريق والرؤية
- **الأسعار**: عرض الباقات الثلاث (مجاني، احترافي، أعمال)
- **التذييل**: روابط سريعة ومعلومات التواصل

### واجهة المحادثة

واجهة المحادثة هي قلب التطبيق:

- **شريط جانبي**: قائمة المحادثات السابقة مع تصنيف زمني
- **حالة السياق**: مؤشر بصري لاستهلاك ذاكرة المحادثة
- **بطاقات القدرات**: اقتراحات ذكية للأسئلة الهندسية
- **دعم Markdown**: عرض الرموز والمعادلات والجداول
- **دعم LaTeX**: عرض المعادلات الهندسية بشكل جميل
- **عرض الكود**: تمييز صياغة الأكواد البرمجية
- **عرض الصور**: تحليل الصور المرفقة
- **مصادر المعلومات**: عرض المصادر المستخدمة في الإجابة
- **تقييم الإجابات**: إعجاب/عدم إعجاب للمساعدة في التحسين
- **إعادة توليد**: إعادة生成 الإجابة بنموذج مختلف

### صفحة المستندات

إدارة قاعدة المعرفة:

- **رفع ملفات**: دعم PDF, DOCX, TXT, XLSX, PPTX, HTML, CSV
- **سحب وإفلات**: واجهة سهلة لرفع الملفات
- **حالة المعالجة**: مؤشرات حية لحالة كل مستند
- **صحة المستند**: تقييم جودة المستند المعالج
- **بحث**: البحث في المستندات
- **فلترة**: حسب النوع والحالة

## ⚙️ لوحة الإدارة

لوحة إدارة شاملة للمشرفين:

### لوحة القيادة (Dashboard)

- **إحصائيات حية**: عدد المستخدمين، الطلبات، التكلفة
- **رسم بياني**: نشاط النظام خلال 15 يوم
- **حالة المزودين**: مؤشرات صحة مزودي الذكاء الاصطناعي
- **إشعارات النظام**: تنبيهات وتحديثات

### إدارة المستخدمين

- **قائمة المستخدمين**: بحث وفلترة وتصفح
- **تعديل البيانات**: الاسم، البريد، الدور، الباقة
- **حذف المستخدمين**: مع تأكيد
- **إحصائيات**: الاستخدام اليومي لكل مستخدم

### إدارة الباقات

- **إنشاء باقات**: اسم، سعر، حد يومي، مميزات
- **تعديل الباقات**: تحديث أي باقة
- **إحصائيات**: عدد المشتركين في كل باقة
- **تفعيل/تعطيل**: التحكم في توفر الباقات

### إعدادات الدفع

- **PayPal**: إعدادات بوابة PayPal
- **Stripe**: إعدادات بوابة Stripe
- **إعدادات عامة**: تفعيل/تعطيل نظام الدفع

### إدارة المستندات RAG

- **رفع مستندات**: من لوحة الإدارة
- **مصادر خارجية**: استيراد من URLs, YouTube, GitHub
- **حزم هندسية**: استيراد datasets هندسية جاهزة
- **عرض الكتل**: تفاصيل كل مستند مع الكتل النصية
- **تصدير/استيراد المتجهات**: نسخ احتياطي للفهرسة
- **إعادة المعالجة**: مع طبقة الترميم الذكي

### قاعدة المعرفة

- **مدخلات معلقة**: مراجعة واعتماد المعرفة المستخرجة
- **مدخلات معتمدة**: المعرفة المعتمدة في النظام
- **استيراد**: من JSON, نص, ملفات, روابط
- **بحث ذكي**: استخراج تلقائي للمعرفة
- **تنظيف التكرارات**: إزالة المعرفة المكررة

### مركز التعلم الذكي

- **مهام بحث**: إطلاق مهام بحث شاملة على الإنترنت
- **تعلم من مصادر**: استيراد معرفة من URLs مباشرة
- **عمق التعلم**: قياسي أو عميق
- **موافقة تلقائية**: نشر مباشر بدون مراجعة
- **سجل المهام**: تتبع جميع مهام التعلم

### مزودي الذكاء الاصطناعي

- **إدارة المفاتيح**: إضافة/حذف/تعديل مفاتيح API
- **أولوية المفاتيح**: ترتيب المفاتيح حسب الأولوية
- **اختبار الاتصال**: التحقق من عمل كل مفتاح
- **مزودين متعددين**: Groq, OpenAI, Google, OpenRouter, NVIDIA, Anthropic, Moonshot, DeepSeek, HuggingFace, Jina
- **إدارة النماذج**: إضافة/تعطيل/حذف النماذج
- **مزامنة**: جلب أحدث النماذج من المزودين

### إعدادات النظام

لوحة إعدادات متقدمة مع أقسام:

1. **النماذج والفهم**:
   - النموذج الافتراضي
   - مهندس الفهم الخلفي (Query Interpreter)
   - Presets جاهزة (متوازن، محافظ، هجومي)

2. **المحادثة العامة**:
   - وضع المحادثة (Chat/Query)
   - حجم الذاكرة
   - درجة الإبداع (Temperature)

## 🧠 نظام الذكاء الاصطناعي

### مهندس الفهم الخلفي (Query Interpreter)

نظام متقدم لفهم نوايا المستخدم:

```json
{
  "intent": "calculation",
  "questionType": "concrete_quantity",
  "needsClarification": true,
  "missingParams": ["length", "width"],
  "extractedParams": {
    "thickness": "20 سم"
  },
  "rewrittenQuery": "احسب كميات الخرسانة لسقف",
  "confidence": 0.92
}
```

### نظام التوجيه (Intent Router)

توجيه الطلبات للمسار المناسب:

| النية | المسار | الوصف |
|-------|--------|-------|
| `knowledge` | RAG + LLM | أسئلة معرفية وشرح |
| `calculation` | Clarification + Solver | حسابات هندسية |
| `design` | Engineering Solver | تحليل وتصميم |
| `chat` | LLM مباشر | محادثة عامة |

### نظام التوضيح (Clarification)

عندما تكون المعطيات ناقصة:

1. **تحديد المعطيات الناقصة**: تحليل ما يحتاجه السؤال
2. **طلب المعلومات**: سؤال المستخدم عن القيم الناقصة
3. **تجميع المعطيات**: جمع الإجابات
4. **تنفيذ الحساب**: إجراء الحساب بالقيم الكاملة

### نظام الكاش الذكي

- **كاش دلالي**: مطابقة الأسئلة المتشابهة معنويًا
- **عتبة التشابه**: قابلة للتعديل (افتراضي 0.7)
- **أقل من 200ms**: استجابة فورية للأسئلة المكررة
- **توفير التكلفة**: تقليل استهلاك API

## 📚 نظام RAG

### دورة حياة المستند

```
رفع ملف → معالجة → تقطيع → تضمين → فهرسة → جاهز للبحث
```

### استراتيجيات التقطيع

| الاستراتيجية | الوصف | الاستخدام |
|--------------|-------|-----------|
| **هرمي** | تقسيم حسب الأقسام ثم الفقرات | المستندات العربية المنظمة |
| **حجم ثابت** | تقسيم بحجم ثابت مع تداخل | النصوص العامة |
| **جملة بجملة** | تقسيم عند نهاية كل جملة | المحادثات والنصوص القصيرة |
| **فقرة بفقرة** | تقسيم عند نهاية الفقرات | المقالات والتقارير |

| **Jina** | jina-embeddings-v3 | 1024 | سريع ومجاني |

### إعادة الترتيب (Reranking)

نظام يحسن دقة النتائج:

2. تحليل النتائج بنموذج خاص

4. إرجاع أفضل النتائج
